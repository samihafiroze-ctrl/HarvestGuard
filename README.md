Harvest Guard – কৃষক সহায়ক

A smart, beginner-friendly web application designed to help farmers in Bangladesh monitor crop conditions, weather forecasts, and essential farming data. The app provides an easy way to track harvest details, costs, notes, and get weather-based insights — all in Bangla, making it accessible to local farmers. Built for EDU HackFest 2025.

Project Description

Harvest Guard is a tech-based agricultural assistant that helps farmers reduce losses, plan effectively, and stay informed.

Main features include:

5-day weather forecast for better crop planning

Crop batch logging (type, harvest date, storage, estimated weight)

Cost tracking (seed, fertilizer, labor, etc.)

Farmer notes for reminders and observations

Market advice (prototype)

Simple Bangla interface

Works using LocalStorage — no backend needed

Fully deployable on Vercel

Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

LocalStorage

Weather API (Open-Meteo or WeatherAPI.com)

Vercel for deployment

Installation / Setup Instructions
1. Download or clone the project You can either: Download ZIP from GitHub and extract it
2. Open the folder Go to the project folder where the files are.
3. Open the app Just double-click index.html. That’s it — the app will open in your browser and work immediately. 4. Everything saves automatically You don’t need any setup. You don’t need any server. LocalStorage saves all your data inside your browser.
